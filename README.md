# Video-Mate
A Multiuser video calling application for Android.<br/><br/>
**Features**
- Accept call
- Send invitation
- Notification for missed calls
- Enable low bandwidth mode
- Raise your hand
- Toggle camera
- Enter tile view
- Chatting
- Recording
- Start live stream
- Select sound device

Developed using Jitsi Meet based upon Web RTC.<br/>
